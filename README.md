     > 三个判断条件
     >
     > - 当前遍历的元素T[i]小于栈顶元素T[st.top()]的情况
     > - 当前遍历的元素T[i]等于栈顶元素T[st.top()]的情况

15. 寻找存在的路径

    > 并查集的三个基本功能：
    >
    > - 寻找根节点，函数：find(int u)，也就是判断这个节点的祖先节点是哪个
